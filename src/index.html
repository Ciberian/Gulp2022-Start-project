Сборка html страницы

Сборка html страницы осуществляется плагином gulp-file-include. Для того чтобы вставить на страницу фрагмент из другого html файла (например содержимое файл head.html), необходимо использовать директиву @\@include('html/head.html', {"title": "Главная страница"}). В фигурных скобках можно указать несколько переменных, значение которых подставится в вёрстку вызываемого файла. В данном случае у вызываемого файла head.html в теге <title>@@title</title> находится имя переменной, вместо которого подставятся слова "Главная страница". В случае если мы будем использовать эту шапку при сборке другой страницы, например при сборке страницы catalog.html то в директиве include мы уже укажем другое значение переменной title. Например { "title": "Каталог" }. И у страницы catalog.html будет <title>Каталог</title>
